# Data Field Professionals Survey Analysis

This project analyzes survey responses from professionals working in the **data field**. It explores trends on **salary satisfaction, work-life balance, career difficulty, and overall happiness**.

## Project Goal
- Understand the experiences and challenges of data professionals.
- Identify factors that influence career satisfaction and happiness.
- Visualize trends for better insights using interactive dashboards.

## Tools Used
- **Power BI Desktop** – for dashboards and visualizations.
- **Excel** – raw survey dataset.

## Folder Structure
- `data/` – contains the raw Excel survey data.
- `dashboard/` – Power BI `.pbix` file.
- `README.md` – project description and instructions.

## How to Use
1. Open `dashboard/DataField_Survey.pbix` in Power BI Desktop.
2. Connect it to `data/survey_raw.xlsx` (if not already embedded).
3. Explore dashboards such as:
   - Average Salary by Role
   - Average Age and Total Count of Participants
   - Average Happiness with Salary and Work Life Balance

## Insights (Summary)
- Generally low level of satisfactoin with Salary and Work Life Balance
- Data Scientists have the highest average salary while Databas Developers have the least
